Instruções
Formato do Arquivo:

O arquivo Excel deve conter as seguintes informações (primeira linha como cabeçalho):

  • Tipo - Tipo de registro (Requisição de Serviço ou Projeção de Gasto)
  • Descrição - Descrição do serviço ou gasto
  • Centro de Atividade - Código do centro de atividade
  • Nome Centro de Atividade - Nome do centro de atividade
  • Valor Planejado - Valor planejado (formato brasileiro: 2.982,99)
  • Valor Realizado - Valor realizado
  • Valor Projetado - Valor projetado
  • Data Requisição - Data da requisição (DD/MM/AAAA)
  • Data Planejada - Data planejada
  • Data Realizada - Data realizada
  • Mês Referência - Mês de referência
  • Ano Referência - Ano de referência
  • Fornecedor - Nome do fornecedor
  • Número Requisição - Número da requisição
  • Número Ordem de Serviço - Número da ordem de serviço
  • Status - Status do registro
  • Observações - Observações gerais
Observações:
  • A primeira linha do arquivo deve conter os cabeçalhos das colunas
  • O sistema identifica automaticamente o tipo de registro baseado no nome da planilha ou colunas
  • O sistema aceita valores monetários no formato brasileiro (ex: 2.982,99) ou formato americano (ex: 2982.99)
  • Datas podem estar em formato DD/MM/AAAA ou formato Excel nativo
  • O sistema processa múltiplas planilhas se o arquivo contiver mais de uma planilha

Nota: O sistema suporta arquivos Excel com diferentes estruturas e tenta detectar automaticamente os campos corretos baseado nos nomes das colunas.
