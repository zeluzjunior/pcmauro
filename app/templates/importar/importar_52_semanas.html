Importe dados das 52 semanas do ano através de arquivo Excel

Formatos aceitos: Excel (.xlsx, .xls, .xlsm)
Quando marcado, apenas registros que ainda não existem na tabela serão importados. Registros duplicados serão ignorados.
Quando marcado, registros existentes serão atualizados com os dados do arquivo. Nota: Se "Apenas adicionar novos registros" estiver marcado, esta opção será ignorada.
Instruções
Formato do Arquivo:

O arquivo Excel deve conter as seguintes colunas:

  • semana: Número ou descrição da semana (ex: "semana 52, 2025", "semana 01")
  • inicio ou início: Data de início da semana
  • fim: Data de fim da semana
Formatos de Data Suportados:

O sistema aceita datas em vários formatos diferentes:

  • Texto com mês: "22 Dezembro 2025", "22 Dec 2025", "22 de Dezembro de 2025"
  • Formato brasileiro: "22/12/2025", "22-12-2025", "22.12.2025"
  • Formato ISO: "2025-12-22", "2025/12/22"
  • Formato curto: "22/12/25", "22-12-25"
  • Objetos de data do Excel: O sistema detecta automaticamente datas do Excel
  • Números Excel: Números de série de data do Excel são convertidos automaticamente

Nota: O sistema tenta automaticamente detectar o formato da data. Se houver problemas, verifique se as datas estão em um dos formatos acima.

Observações:
  • A primeira linha do arquivo deve conter os cabeçalhos das colunas
  • O campo "semana" é obrigatório
  • As datas são opcionais, mas recomendadas
  • Registros com a mesma "semana" serão considerados duplicados
