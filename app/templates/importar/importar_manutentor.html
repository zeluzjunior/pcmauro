Importar Manutentor

Importe dados de manutentores através de arquivo Excel ou CSV

Selecionar Arquivo
{% csrf_token %}
Formatos aceitos: Excel (.xlsx, .xls, .xlsm) ou CSV (.csv)
Instruções de Importação
Formato do Arquivo

O arquivo Excel ou CSV deve conter as seguintes colunas (primeira linha como cabeçalho):

Coluna Descrição Obrigatório Valores Aceitos
MATRICULA ou CADASTRO Matrícula do manutentor Sim Texto único
NOME Nome completo do manutentor Não Texto
CARGO Cargo do manutentor Não Texto
HORARIO_INICIO Horário de início do trabalho Não HH:MM ou HH:MM:SS
HORARIO_FIM Horário de fim do trabalho Não HH:MM ou HH:MM:SS
TEMPO_TRABALHO Tempo de trabalho Sim Texto
TURNO Turno de trabalho Sim Turno A, Turno B, Turno C
LOCAL_TRAB Local de trabalho Sim Industria, Frigorífico, Civil, Indefinido, ETE/ETA, Utilidades, Manutenção
Observações Importantes
  • O sistema aceita variações nos nomes das colunas (maiúsculas/minúsculas, com ou sem acentos)
  • Para arquivos CSV, use ponto e vírgula (;) como delimitador
  • A primeira linha deve conter os cabeçalhos das colunas
  • Campos obrigatórios: MATRICULA (ou CADASTRO), TEMPO_TRABALHO, TURNO, LOCAL_TRAB
  • Se a matrícula já existir e a opção "Atualizar registros existentes" estiver marcada, o registro será atualizado
Exemplo de Estrutura
MATRICULA;NOME;CARGO;HORARIO_INICIO;HORARIO_FIM;TEMPO_TRABALHO;TURNO;LOCAL_TRAB
12345;João Silva;Mecânico;08:00;17:00;8 horas;Turno A;Industria
12346;Maria Santos;Eletricista;07:00;16:00;8 horas;Turno B;Frigorífico
