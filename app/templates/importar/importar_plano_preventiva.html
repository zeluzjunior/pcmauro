Importar Plano Preventiva

Importe dados de plano de manutenção preventiva através de arquivo

Selecionar Arquivo
{% csrf_token %}
Formatos aceitos: Excel (.xlsx, .xls, .xlsm) ou CSV (.csv)
Quando marcado, apenas registros que ainda não existem na tabela serão importados. Registros duplicados serão ignorados.
Quando marcado, registros existentes serão atualizados com os dados do arquivo. Nota: Se "Apenas adicionar novos registros" estiver marcado, esta opção será ignorada.
Instruções de Importação
Formato do Arquivo

O arquivo deve conter as seguintes colunas (primeira linha como cabeçalho), separadas por ponto e vírgula (;):

Coluna Descrição Obrigatório
Unidade Código da Unidade Não
Nome Unidade Nome da Unidade Não
Setor Código do Setor Não
Descrição Setor Descrição do Setor Não
Atividade Código da Atividade Não
Máquina Código da Máquina Recomendado
Descrição Máquina Descrição da Máquina Recomendado
Nº Patrimônio Número do Patrimônio Não
Plano Número do Plano Não
Descrição Plano Descrição do Plano (ex: "PREVENTIVA") Não
Sequência Manutenção Sequência da Manutenção Não
Data Execução Data de Execução (formato: DD/MM/YYYY) Não
Quantidade Período Quantidade do Período em dias Não
Sequência Tarefa Sequência da Tarefa Não
Descrição Tarefa Descrição da Tarefa de Manutenção Não
Funcionário Código do Funcionário Responsável Não
Nome Funcionário Nome do Funcionário Responsável Não
Formato Aceito
  • CSV: Arquivo delimitado por ponto e vírgula (;) com encoding latin-1 ou utf-8
  • Excel: Arquivos .xlsx, .xls ou .xlsm
  • A primeira linha deve conter os cabeçalhos das colunas
  • Linhas vazias serão automaticamente ignoradas
Importante: O sistema tentará vincular automaticamente os planos preventiva às máquinas existentes no banco de dados através do código da máquina. Se a máquina não existir, o plano será importado sem vínculo.
Nota: O sistema identifica registros duplicados pela combinação de: Código da Máquina + Número do Plano + Sequência Manutenção + Sequência Tarefa
